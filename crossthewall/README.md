# Cross the Wall — juego 2D

Experiencia interactiva del conflicto mapuche–Estado calibrada con ELRI.
Zona rural (mapuche) ↔ muro ↔ zona urbana (no indígena). Cuatro olas,
~30 s cada una, con shock contrafactual del estallido 2019.

## Estructura

```
crossthewall/
├── core/    # Motor del juego (mapa, NPCs, ABM, conversaciones)
├── ui/      # HUD e interfaz
└── README.md
```
